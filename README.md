# 📊 FolhaRH Pro – Relatório Executivo de Folha de Pagamento  
**Análise de Custos DP – App Digital**  
💰 Valor: R$ 197,00  

## 🎯 Objetivo  
Apresentar de forma clara e rápida os **custos totais da folha de pagamento da empresa** para a diretoria.  

## 🚀 O que é  
FolhaRH Pro gera relatórios executivos de folha em minutos.  
A IA importa o PDF e preenche todos os campos automaticamente.  
Em poucos minutos você tem o relatório completo pronto para apresentar à diretoria.  

## 🏢 Para quem é  
- Analistas e gestores de DP/RH  
- Escritórios de contabilidade  
- Empresas com matriz + filial (2 CNPJs)  
- Profissionais que apresentam custos de folha mensalmente  

## 🤖 Diferencial  
- Importação automática de PDF com IA  
- Compatível com qualquer sistema (Domínio, Senior, ADP, Totvs, Questor...)  
- Extrai bruto, líquido, descontos, INSS, FGTS, IRRF e mais  
- Resultado em segundos, sem digitação manual  

## 📊 Módulos principais  
- Resumo da Folha (bruto, líquido, descontos, funcionários)  
- Encargos Patronais (INSS, FGTS, IRRF, RAT/FAP, terceiros)  
- Conferência de Guias (folha × guia)  
- Benefícios (iFood, Caju, Unimed, planos de saúde via CSV)  
- Movimentações (admissões/demissões)  
- Prestadores PJ (contratos e valores mensais)  
- Banco de Horas (saldo e passivo financeiro)  
- Histórico Anual (até 24 meses)  
- Comparativo entre 2 CNPJs  
- Relatório PDF executivo pronto para diretoria  

## 💾 Segurança dos dados  
- Dados 100% locais no navegador  
- Nenhuma informação enviada a servidores externos  
- Backup/restauração em JSON com 1 clique  
- Histórico de até 24 meses disponível
